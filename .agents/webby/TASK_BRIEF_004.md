### Skill 2: PlatformFormatter
**Purpose:** Adapt content for each platform's constraints

**Platforms:**
| Platform | Constraint | Formatter |
|----------|------------|-----------|
| Twitter | 280 chars/thread | Thread builder |
| LinkedIn | 3,000 chars | Long-form adapter |
| TikTok | 60-90 sec | Script generator |
| YouTube | 3-15 min | Trailer/expander |
| Instagram | 2,200 chars + carousel | Visual prioritizer |
| Substack | 1,500+ words | Full essay |

**Implementation:**
```javascript
function formatForPlatform(content, platform) {
  switch(platform) {
    case 'twitter': return buildThread(content, 15);
    case 'linkedin': return longForm(content, 300);
    case 'tiktok': return shortScript(content, 60);
  }
}
```

---

### Skill 3: CrossDistributor
**Purpose:** Publish to multiple platforms with tracking

**Supported Platforms:**
- Twitter/X API
- LinkedIn API
- Reddit API
- Discord Webhooks
- Substack (email + web)

**Workflow:**
1. Queue content per platform
2. Schedule optimal times
3. Post with tracking IDs
4. Log URLs to dashboard
5. Monitor engagement

**Implementation:**
```javascript
async function distribute(content, platforms) {
  for (const platform of platforms) {
    const formatted = formatFor(content, platform);
    const result = await postTo(platform, formatted);
    logToDashboard(platform, result.url);
  }
}
```

**Outputs:**

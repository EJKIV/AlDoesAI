# Al Does AI — Content Templates

**Standard formats for each platform.**

---

## Twitter/X — Daily Prompt Thread

### Format: 5-7 tweets

**Tweet 1 (Hook):**
```
Day [N] of building with an AI business partner:

[Surprising result in one line]

Here's exactly how we did it ↓
```

**Tweet 2 (Setup):**
```
The prompt:
"[First 100 chars of prompt]..."

Full prompt: [GitHub link]
```

**Tweet 3 (What happened):**
```
What came back:
- [Specific output 1]
- [Specific output 2]
- [Specific output 3]

Time: [X minutes]
```

**Tweet 4 (The result):**
```
The outcome:

[Specific, tangible result with numbers]
```

**Tweet 5 (Investment):**
```
The investment:
- Time: [X minutes]
- Tools: [Cost]
- Total so far: [$X invested, $Y revenue]
```

**Tweet 6 (Learnings):**
```
What we learned:
1. [Specific insight]
2. [Specific insight]
3. [Specific insight]
```

**Tweet 7 (CTA):**
```
Tomorrow: [Next day preview]

Follow along → Every prompt, every day

[Link to newsletter/substack]
```

## Substack — Weekly Deep Dive

### Format: 1,500-2,000 words

**Sections:**
1. **The Week in Numbers** — 30-second summary
2. **Daily Prompts** — Each day with full context
3. **The Stack** — Tools added/changed
4. **Financials** — Investment vs revenue
5. **What Failed** — The messy parts
6. **What Worked** — Measurable wins
7. **Ask Me Anything** — Community Q&A

---

## YouTube — Deep Dive Video

### Format: 10-15 minutes

**Structure:**
1. **Hook** (0:00-0:30): "This week my AI built..."
2. **Context** (0:30-1:00): Where we started
3. **The Prompt** (1:00-2:00): Screen share of conversation
4. **The Result** (2:00-8:00): Demo what was built
5. **The Numbers** (8:00-9:00): Investment vs outcome
6. **Lessons** (9:00-10:00): Key takeaways
7. **Next Week** (10:00-11:00): What's coming
8. **CTA** (11:00-12:00): Subscribe, follow, join

---

## Quick Win Posts (TikTok/Reels)

### Format: 30-60 seconds

**Template:**
```
[Text on screen]: "I asked my AI to [task]"

[Screen recording]: Fast-forward of AI working

[Text on screen]: "[Time] later..."

[Screen recording]: The result

[Text on screen]: "Investment: $[cost] + [time]"

[Text on screen]: "Want this? Link in bio"
```

## Visual Style Guide

### Colors
- **Primary:** #6B46C1 (Deep Purple)
- **Secondary:** #22D3EE (Electric Cyan)
- **Background:** #0F0F0F (Near Black)
- **Text:** #F5F5F5 (Warm White)
- **Accent:** #F97316 (Orange)

### Fonts
- **Headers:** Inter Bold
- **Body:** Inter Regular
- **Code:** JetBrains Mono

### Graphics
- **Prompt cards:** Dark background, cyan border, timestamp
- **Result cards:** Dark background, purple border, checkmark
- **Investment cards:** Dark background, orange border, dollar sign
- **Progress bars:** Animated, showing delegation ratio

### Imagery
- **Human side:** Jim's workspace, coffee, notes
- **AI side:** Terminal, code, dashboards
- **Together:** Split screen, collaboration

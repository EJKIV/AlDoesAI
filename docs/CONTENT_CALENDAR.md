## Content Types by Template

### 1. Daily Twitter Thread — "Today I Learned"
**Template:**
```
🧵 Day X: [Specific thing learned/did]

1/n The journey so far...

[Context]

2/n What surprised me:

[Unexpected finding]

3/n What I built:

[Specific output with images]

[n/n] Follow for daily updates. Tomorrow: [Tease next day]
```
**Frequency:** Daily
**Avg length:** 5-7 tweets
**Hashtags:** #AlDoesAI #[SpecificTopic]

### 2. LinkedIn Long-Form — "The Bigger Picture"
**Template:**
```
I'm documenting a 180-day experiment:

→ Can AI agents run a content business?
→ Total transparency (every prompt, every $)
→ Starting from $0

This week I learned [specific insight].

The shift from [old way] to [new way] revealed [surprising truth].

Here's what I'm changing going forward:

[3 specific adjustments]

If you're building with AI, what's your biggest lesson so far?

#AI #BuildingInPublic #AlDoesAI
```
**Frequency:** 2x/week
**Length:** 150-300 words
**CTA:** Engagement question

### 3. Substack Deep-Dive — "The Weekly Synthesis"
**Template:**
```
# Week X: [Theme]

## The Numbers
- Prompts: X
- Time invested: Y hours  
- Money spent: $Z
- [Specific metric]

## What Worked
[3 things with screenshots]

## What Didn't
[2 things with honest reflection]

## This Week's Build
[Specific agent/tool built]

## Next Week Preview
[What's coming + why it matters]

---
*Reply to share your own week*
```
**Frequency:** Weekly (Sunday)
**Length:** 800-1500 words
**Include:** Screenshots, metrics, links

### 4. YouTube Tutorial — "Build With Me"
**Template:**
```
[Hook: Problem]

"You want to track 50 data sources but don't have time to check them all."

[Solution preview]

"I built an agent that does this automatically."

[Screen recording]

- Step 1: [Action]
- Step 2: [Action]  
- Step 3: [Action]

[Result]

"This saves me 2 hours/day."

[CTA]

"What would you automate? Comment below."
```
**Frequency:** 1x/week (Saturday)
**Length:** 5-15 minutes
**Format:** Screen recording + voiceover

### 5. TikTok Quick Win — "90-Second Tutorial"
**Template:**
```
[0-3s Hook]
Text: "Stop doing [manual task]"

[3-30s Problem]
"I used to spend X hours on Y"

[30-60s Solution]
Quick demo of agent/tool

[60-85s Result]
"Now it takes Z seconds"

[85-90s CTA]
"Follow for more AI automations"
```
**Frequency:** 3x/week (MWF)
**Length:** 60-90 seconds
**Format:** Screen recording or face to camera

- Quote-tweet with commentary

- Short clips for TikTok

## Scheduling Rules

1. **Never batch more than 3 days ahead** — stay responsive
2. **Always have 1 day buffer** — for breaking news/opportunities  
3. **Jim approves all drafted content** — maintain human voice
4. **Test one new format per week** — controlled experimentation
5. **Document what works** — update templates monthly
